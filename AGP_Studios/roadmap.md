# AGP Studios Roadmap (high level)

Overview
- Goal: transform AGP_Studios into a Steam-like client + development studio with AI assistant "Ashat".
- Timeline: phased (Discovery → MVP Launcher → Remote Store → Multiplayer → Marketplace & AI → Scale).

Phases
- Phase 0: Discovery & Alignment (1–2 wks)
  - Tech stack decision, architecture doc, OpenAPI skeleton.

- Phase 1: MVP Launcher + Local Server Builder (6–10 wks)
  - Launcher install/update/launch.
  - Server scaffolding wizard.
  - Local AGP_AI integration (assistant for scaffolding).

- Phase 2: Remote Hosting & Store (8–12 wks)
  - CMS APIs, asset hosting, remote catalog.
  - Launcher download & update from CMS.

- Phase 3: Multiplayer & Social (12–16 wks)
  - Matchmaking, sessions, friends/invite flows.

- Phase 4: Creator Tools & Marketplace (ongoing)
  - Publishable assets, plugin SDK, AI-assisted publishing.

- Phase 5: Scale & Monetize (long term)
  - Payments, global scale, monitoring, moderation.

Next steps
- Decide desktop UI tech (WPF, MAUI, Blazor or Electron).
- Create project board with the Phase 1 tasks.
